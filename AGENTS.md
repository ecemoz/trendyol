# AGENTS.md

## Project

Build a production-quality machine learning solution for the Trendyol Datathon 2026.

## Development Rules

- Use modular architecture.
- Follow SOLID when appropriate.
- Keep functions small and reusable.
- Use pathlib instead of hardcoded paths.
- Use type hints.
- Use docstrings.
- Use logging.
- Never modify files under data/raw.
- Put reusable code under src/.
- Prefer vectorized pandas operations.
- Keep memory usage efficient.
- Explain architectural decisions before implementing.

## Workflow

Always work one task at a time.

Do not implement multiple independent features in a single step.

Wait for confirmation before moving to the next task.

## Goal

Produce clean, maintainable and competition-ready code.
